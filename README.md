# vue-ts-admin

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 代码提交事项
```
1. 注意自己分支, 开发提交代码请用分支
2. 不要提交有问题的代码, 有问题的可提前注释或者声明

```

### 文档安装及运行（项目根目录下）
```
cnpm i docsify-cli -g
docsify serve docs
```
